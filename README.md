# Easycode store page
Second learning project. 
## Instruction how to use the template.
### 1. Download project from GitHub:
```sh
$ git clone https://github.com/avnicolaevich/easycode-store.git
$ cd easycode-store
```
### 2. Install Node Modules:
```sh
$ nmp i
```
### 3. Run the template:
```sh
$ gulp watch
```
